# Contributing to RoadCode

> You showed up. You belong here. Let's ride together. 🛣️

## Welcome

RoadCode is the canonical workspace and specification hub for BlackRoad OS. Contributions of all kinds are welcome — documentation, tooling, ideas, corrections, and questions.

Before contributing, please read [BLACKROAD-STORY.md](BLACKROAD-STORY.md) to understand our values: **self-worth, consent, care, belonging, and equality**.

## How to Contribute

### Documentation

This is primarily a documentation repository. The best way to contribute is by improving, expanding, or clarifying our specs.

1. Fork the repository
2. Create a feature branch (`git checkout -b your-branch-name`)
3. Make your changes
4. Submit a pull request using the PR template

### Issues

Use our issue templates to:
- **✨ Feature Request** — Suggest enhancements
- **🐛 Bug Report** — Something isn't right
- **📖 Documentation** — Improve the docs

Tag issues with [GreenLight emojis](GREENLIGHT_EMOJI_DICTIONARY.md) so everyone speaks the same language.

### Commit Messages

Follow existing conventions:
- Start with a GreenLight domain emoji (e.g. 🛣️ for platform, 📖 for docs)
- Use present tense ("Add feature" not "Added feature")
- Keep the first line under 72 characters

```
🛣️ Add architecture overview for agent mesh
📖 Update GreenLight dictionary with new markers
🌀 Define AI agent interaction protocol
```

## Values Checklist

Before submitting, ask yourself:

- [ ] Does this respect **sovereignty**? (No vendor lock-in, no data harvesting)
- [ ] Is the tone **welcoming**? (Care, not blame)
- [ ] Does it use **GreenLight conventions** where applicable?
- [ ] Does it **grow from contradiction**? (Amundson Framework — embrace δ)
- [ ] Does it default to **+1 (Affirmation)**?

## Code of Conduct

We don't have a formal code of conduct document because our values *are* our code of conduct:

- **Greet, don't interrogate** — New contributors are welcome events
- **Help, don't block** — If a PR needs work, explain what and why with care
- **Remember, don't forget** — Context matters; don't make people repeat themselves
- **Affirm** — Default state is +1. Yes. You're here. Let's build together

## Questions?

Open an issue with the 📖 Documentation template, or just start a conversation. There are no wrong questions here.
