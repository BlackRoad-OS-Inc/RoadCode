## Connection Standard

Every connected org has:
- `RoadCode/README.md` — links back to BlackRoad-OS-Inc
- `RoadCode/CONNECTOR.md` — machine-readable JSON link
- `.github/profile/README.md` — org profile with parent reference

## Verification

```bash
# Verify all connections
for org in BlackRoad-OS BlackRoad-Studio BlackRoad-Archive BlackRoad-Interactive BlackRoad-Security BlackRoad-Gov BlackRoad-Education BlackRoad-Hardware BlackRoad-Media BlackRoad-Foundation BlackRoad-Ventures BlackRoad-Cloud BlackRoad-Labs BlackRoad-AI Blackbox-Enterprises; do
  gh api "repos/$org/RoadCode/contents/CONNECTOR.md" -q '.name' 2>/dev/null && echo "✓ $org" || echo "✗ $org"
done
```
